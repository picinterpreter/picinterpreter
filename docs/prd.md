# 图语家已有功能清单

本文档整理当前项目已经具备的主要功能，按用户可见功能、数据能力、后端能力和工程工具分组。

## 一、核心沟通功能

### 表达模式

- 按分类浏览图片卡片。
- 点击图片加入表达暂存区。
- 支持已选图片的删除、清空、拖拽排序。
- 根据已选图片生成候选中文句子。
- 候选句自动轮流语音播报。
- 支持停止播报、全部重播、点击单句重播。
- 可将当前候选句收藏为常用表达。
- 完成表达后自动记录到对话历史。

### 接收模式

- 照护者可输入一句话，将文字转换为图片序列。
- 支持浏览器语音输入，将语音识别文本后自动转图片。
- 文本匹配支持精确匹配、同义词匹配、词库同义词、包含匹配。
- 匹配率低或有未匹配词时，可调用后端 AI 辅助重分词。
- 转换结果支持删除图片、左右调整顺序、手动换图。
- 支持搜索图库并替换某个词对应的图片。
- 支持全屏展示图片序列给患者。
- 接收流程完成后可记录到对话历史。

### 语音播报

- 使用浏览器 Web Speech TTS。
- 支持中文语音自动选择。
- 支持设置语速。
- 支持选择具体语音。
- 支持语音试听。
- TTS 不可用时有文字提示或大字兜底展示。

## 二、图片与图库功能

### 分类图库

- 本地内置分类与图片种子数据。
- 首次加载或种子版本升级时自动导入 IndexedDB。
- 分类支持图标、排序、隐藏/显示。
- 支持虚拟分类“最近”，展示最近使用图片。

### 图片搜索

- 表达模式下支持跨分类搜索图片。
- 搜索支持中文标签与同义词。
- 搜索输入带防抖，减少本地数据库扫描频率。

### 图片使用统计

- 点击图片后更新使用次数。
- 记录最近使用时间。
- 分类内图片可按使用频次优先显示。
- 最近使用列表基于 `lastUsedAt` 展示。

### 下一词建议

- 暂存区为空时，显示最近使用图片。
- 暂存区非空时，基于最后一个已选图片的分类推荐高频相关图片。
- 自动排除已选图片，减少重复选择。

### 分类链接管理

- 支持为某个分类链接其他分类。
- 浏览当前分类时，会自动显示被链接分类的图片。
- 支持添加、移除分类链接。
- 内置环路检测，避免分类链接形成循环。
- 种子数据重导时会保留用户维护的分类链接。

## 三、AI 与句子生成

### 在线 AI 句子生成

- 前端通过 Next.js 后端接口调用 AI。
- API Key、Base URL、Model 只在服务端环境变量中读取。
- 支持 OpenAI-compatible `/chat/completions` 接口。
- 生成候选句时会传入图片标签。
- 可注入近期对话上下文，让句子更连贯。
- 可注入图库词汇提示，优先使用图库中存在的词。

### 离线模板兜底

- 未配置 AI 或 AI 调用失败时，自动退回本地模板生成。
- 候选面板会提示当前使用离线模式。
- 保证核心表达流程不依赖网络。

### AI 辅助重分词

- 接收模式中，当规则匹配效果较差时，可请求 AI 重新拆分句子。
- AI 会结合图库词汇，把输入文本映射成更容易匹配图片的词序列。
- AI 返回结果更好时替换规则匹配结果，否则保留原结果。

### AI 状态检测

- 设置页可读取后端 AI 配置状态。
- 显示当前模型和 Base URL。
- 支持测试后端 AI 连接。

## 四、常用表达功能

### 收藏短语

- 表达完成后可收藏候选句。
- 收藏短语可关联当时使用的图片序列。
- 支持手动添加自定义短语。
- 支持重命名短语。
- 支持删除短语。
- 支持一键播报收藏短语。
- 播报时会更新使用次数和最近使用时间。

### 常用语快捷栏

- 表达模式顶部显示使用频率最高的常用表达。
- 最多展示 6 条。
- 点击即可立即播报。
- 无收藏时不显示，不占布局空间。

### 常用语导入导出

- 支持将常用表达导出为 JSON。
- 支持从 JSON 导入常用表达。
- 导入时会跳过已存在短语。
- 会提示新增数量、跳过数量，以及缺失图片 ID 数量。

## 五、对话历史功能

### 对话记录

- 自动记录表达模式和接收模式的使用事件。
- 表达记录包含图片、候选句、最终选中句。
- 接收记录包含原始输入文本和匹配图片。
- 记录按会话分组。
- 30 分钟无活动后自动开启新会话。

### 历史管理

- 抽屉中展示最近 100 条记录。
- 支持收藏/取消收藏单条记录。
- 支持删除单条记录。
- 支持清空全部记录。
- 表达记录支持重新播报。
- 支持导出全部对话记录为文本文件。

## 六、紧急求助功能

### 全屏紧急面板

- 可从顶部入口打开。
- 使用红色高优先级界面，与普通功能明显区分。
- 大按钮设计，适合触控和行动不便用户。

### 预设紧急短语

- 帮帮我。
- 我很痛。
- 请叫医生。
- 我害怕。
- 我不舒服。
- 请安静。
- 我要喝水。
- 我要上厕所。

### 紧急播报兜底

- 点击短语立即语音播报。
- 如果语音不可用，会显示大字文本覆盖层，方便直接展示给照护者。

## 七、账号与云同步

### 匿名设备身份

- 首次打开会自动 bootstrap 匿名设备身份。
- 使用安装 ID 绑定本机设备。
- 服务端通过 HttpOnly Cookie 维护设备/会话状态。

### 账号注册与登录

- 支持用户名、密码、手机号注册。
- 支持用户名密码登录。
- 支持退出登录。
- 设置页显示当前登录状态、用户名、脱敏手机号。

### 本地数据并入账号

- 匿名状态下如果检测到本机已有表达记录或收藏，登录/注册前可选择并入当前账号或放弃本机数据。
- 注册或登录后会完成同步身份切换。

### 表达记录与常用语同步

- 当前云同步覆盖 `expressions` 和 `saved_phrases`。
- 本地变更写入 `syncOutbox`。
- 支持 push 本地变更到 MySQL。
- 支持 pull 服务端增量变更。
- 支持增量游标 `lastPulledChangeId`。
- 支持在线、页面重新可见时自动触发同步。
- 有待同步本地变更时，拉取远端变更会避开本地冲突记录。

## 八、本地存储与离线能力

### IndexedDB 本地数据库

- 使用 Dexie 管理本地数据。
- 存储分类、图片、表达记录、常用表达、文本转图片结果、同步队列、同步状态。
- 用户数据不会因为种子数据重导而清空。

### 本地设置持久化

- 语速。
- 首选语音。
- 高对比度模式。
- 字体大小。
- 图片网格列数。
- 隐藏分类。
- 当前会话状态。
- 首次引导完成状态。

### PWA 支持

- 提供 `manifest.json`。
- 支持独立应用显示模式。
- 有 Service Worker 注册组件。
- 默认通过环境变量控制是否启用 Service Worker。
- 有新版本更新横幅，可提示用户刷新更新。
- 当前默认关闭 Service Worker，并会清理旧缓存，避免旧资源残留。

## 九、无障碍与易用性设置

### 显示设置

- 图片网格支持 2 列、3 列、4 列。
- 字体支持正常、大、超大。
- 支持高对比度模式。
- 支持隐藏患者不常用分类，简化界面。

### 触控友好

- 主要按钮和操作控件使用较大的点击区域。
- 多数弹窗支持 Escape 关闭。
- 紧急求助面板支持焦点管理和键盘关闭。
- 多个区域带有 `aria-label`、`aria-live`、`role` 等辅助属性。

### 首次使用引导

- 首次打开自动显示引导。
- 说明图语家的用途、三步表达流程、AI/离线模式。
- 设置中可重看引导。

## 十、开发与调试工具

### 匹配验证工具页 `/debug`

- 输入文字后展示分词结果。
- 展示每个 token 的图片匹配结果。
- 显示匹配方式：精确、同义词、词库、包含、未匹配。
- 显示匹配率、耗时、分词数、输入字符数。
- 提供常见测试句。

### ARASAAC 导入工具页 `/import`

- 从本地词库批量搜索 ARASAAC 图片。
- 显示导入进度。
- 标记找到、未找到、错误状态。
- 支持停止导入。
- 支持导出新的 `pictograms.json`。
- 未找到图片时保留占位 SVG 路径。

### 种子数据与脚本

- 内置 `categories.json` 和 `pictograms.json`。
- 提供 ARASAAC 缺失图片抓取脚本。
- 提供 CBoard 转换脚本。
- 提供图片清理脚本。
- 提供图标生成脚本。

## 十一、后端接口

### AI 接口

- `GET /api/ai/health`：检查 AI 配置状态。
- `POST /api/ai/sentences`：生成候选句。
- `POST /api/ai/resegment`：AI 辅助重分词。

### 客户端身份与同步

- `POST /api/client/bootstrap`：注册/恢复匿名设备身份。
- `POST /api/sync/push`：推送本地表达记录和常用语变更。
- `GET /api/sync/pull`：拉取服务端增量变更。

### 账号接口

- `POST /api/auth/register`：注册。
- `POST /api/auth/login`：登录。
- `GET /api/auth/me`：获取当前账号状态。
- `POST /api/auth/logout`：退出登录。

- 使用 Next.js App Router。
- 前端 React 19 + TypeScript。
- Tailwind CSS 4。
- Zustand 状态管理。
- Dexie 本地数据库。
- Prisma + MySQL 服务端数据层。

- 使用 Vitest。
- 已有工具函数、同步逻辑、Web Speech、服务端 auth 等测试文件。
- 覆盖文本分词、图片匹配、短语导入导出、分类链接、AI 重分词、NLG 上下文等核心逻辑。

### 部署

- 支持 `npm run build` / `npm run start`。
- 支持 Prisma generate / db push。
- 提供阿里云部署脚本。
- 提供 GitHub Actions 部署说明。
- 支持 Next.js standalone 输出。
