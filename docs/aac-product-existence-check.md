# AAC 产品存在性核查

更新时间：2026-05-06

本文件核查本地 Logseq 笔记 `AAC产品.md` 中提到的 AAC 产品是否真实存在。  
本次只核查“是否能找到可靠公开来源证明产品存在”，不等同于确认以下能力：

- 是否完全免费
- 是否支持中文
- 是否支持 Android
- 是否支持网页端
- 是否支持云同步
- 是否适合成年失语症患者

这些能力需要后续单独做功能核查。很多 AI 推荐会把“存在”与“免费 / Web / 云同步”混在一起，正式文档不应直接照抄。

## 5. 与核心图库调研的关系

这份核查表不是核心图库调研的替代品。

当前核心图库调研已经覆盖：

- Cboard
- AsTeRICS Grid / AsTeRICS AAC Data
- Proloquo2Go / Crescendo
- TD Snap
- TouchChat / WordPower
- LAMP Words for Life
- Avaz AAC
- Widgit Health Bedside Messages
- Lingraphica

但 Logseq 文档里大量产品尚未进入核心图库调研，例如：

- Pictalk AAC
- SweetSuite AAC / CoughDrop
- Weave Chat AAC
- Jellow
- SymboTalk
- QVoice
- Picto4me
- CardChat
- 懂你-失语辅助软件
- AAC 好沟通
- Voice4u AAC
- MaTalk AI

这些产品是否值得补进 `aac-core-library-survey.md`，应取决于它们是否能提供以下参考价值：

- 是否有公开可观察的核心词表或沟通板结构
- 是否面向成年失语症或中文用户
- 是否支持自定义照片、场景板、快速短语
- 是否能启发图语家的核心图库、交互结构或数据模型

## 6. 后续建议

下一步不建议继续只堆产品名。更有价值的是做第二轮功能核查，字段可以是：

| 字段 | 说明 |
|---|---|
| 平台 | Web / Android / iOS / PWA |
| 价格 | 免费 / 试用 / 订阅 / 买断 |
| 云同步 | 无 / 手动备份 / 账号同步 / 实时同步 |
| 是否开源 | 是 / 否 / 部分 |
| 是否支持中文 | 界面 / TTS / 词库 / 图符标签 |
| 图符来源 | ARASAAC / SymbolStix / 自有图库 / 用户上传 |
| 是否适合成年失语症 | 儿童倾向 / 成人可用 / 失语专门 |
| 对图语家的参考价值 | 核心词表 / 场景板 / 自定义照片 / 同步 / AI |

功能核查完成后，再决定哪些产品补进核心图库调研文档，哪些只留在产品存在性清单中。
