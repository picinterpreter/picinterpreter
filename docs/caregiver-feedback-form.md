# 图语家家属 / 照护者反馈表 · Caregiver Feedback Form

这是一份适合家属、照护者、康复老师或试用志愿者使用的反馈表模板。

目标不是收集隐私，而是帮助图语家理解：

- 哪些沟通场景最常见
- 哪些图片容易看不懂
- 哪些词最容易匹配错
- 哪些功能必须离线可用

可以把这份表：

- 复制到问卷工具
- 复制到飞书 / 腾讯文档 / Google Form
- 用作访谈提纲
- 用作 issue / discussion 的匿名整理模板

## 使用前说明

请先向参与者说明：

- 这是公益开源产品反馈，不是医疗诊断或治疗服务
- 不需要提供病历、电话、住址、身份证号等隐私
- 不需要上传患者正脸照片、病历照片或家庭住址信息
- 如果愿意，可以只提供匿名反馈

## 一、基本情况

以下信息只保留对产品有帮助的最少内容。

1. 你的身份是？
   - 家属
   - 日常照护者

   - 志愿者 / 观察者

2. 你主要照护或接触的对象是？
   - 失语患者 / 中风患者
   - 其他需要图片辅助沟通的人

3. 你通常在哪些场景中需要沟通辅助？
   - 家里
   - 医院
   - 康复机构
   - 外出
   - 其他

4. 你更常使用什么设备？
   - 手机
   - 平板
   - 电脑
   - 不固定

## 二、最常见的沟通困难

5. 平时最难沟通的 3 到 5 件事情是什么？

示例：

- 吃饭喝水
- 上厕所
- 身体哪里不舒服
- 想见谁
- 想表达情绪
- 外出 / 回家 / 休息

6. 患者最常想表达的话，大概有哪些？

请尽量写成短句或口语，例如：

- 我想喝水
- 我头疼
- 我想回家
- 太热了
- 帮帮我

7. 家属或照护者最常说、但患者不一定听得懂的话有哪些？

例如：

- 你要不要吃饭
- 我们先去厕所再吃药
- 等一下医生会来

## 三、图片理解与匹配问题

8. 你觉得患者更容易理解哪类图片？
   - 卡通 / 图标
   - 真实照片
   - 家里的真实物品照片
   - 不确定

9. 有没有哪类图片特别容易误解？

可以举例：

- 表情图不容易分辨
- 身体部位图太抽象
- 动作图和物品图容易混

10. 你试用时有没有遇到“说的话配错图片”的情况？

如果有，请尽量写：

- 原话
- 系统匹配成了什么图
- 你本来期望看到什么图

示例：

- 原话：开心
- 错误匹配：开心果
- 期望：开心 / 高兴的表情

11. 有没有哪些重要概念目前应该优先补图？

例如：

- 常见身体症状
- 家庭成员
- 医院场景
- 常吃的中式食物
- 常见护理动作

## 反馈整理模板

如果要把一份反馈匿名整理进 issue / discussion，建议使用下面格式：

```md
## Anonymous caregiver feedback

- Scenario: home / hospital / rehab
- Device: phone / tablet / desktop
- Top communication needs:
  - 喝水
  - 上厕所
  - 哪里疼
- Frequent misunderstanding:
  - 原话：开心
  - 错误匹配：开心果
  - 期望图片：开心表情
- Offline must-work features:
  - 患者选图表达
  - 常用短语
  - 语音播报
- Priority requests:
  - 更多身体部位和疼痛相关图
  - 更容易找到家庭成员和日常食物
```

## 维护者整理时的注意事项

- 去掉姓名、电话、地址、病历和可识别身份的信息
- 如果反馈来自单个家庭，不要保留可回溯的详细故事
- 只提炼和产品设计、词库、交互、离线能力相关的信息
- 涉及患者试用时，优先记录照护者观察，不夸大结论

如果你需要更完整的访谈流程，请同时阅读 [`docs/user-research-playbook.md`](user-research-playbook.md)。
